## Quick Start

1. **Clone Repository**
   ```bash
   git clone https://github.com/soheiloliaei/Fusion-v12.2.git
   cd Fusion-v12.2
   ```

2. **Launch System**
   ```bash
   ./Launch_Fusion_V12.2.command
   ```

3. **Enable Quality System**
   ```python
   export FUSION_QUALITY_ENABLED=true
   ```

4. **Configure Metrics**
   ```json
   {
       "quality_metrics": {
           "minimum_threshold": 0.95,
           "confidence_required": 0.95
       }
   }
   ```
